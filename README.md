# Palmetto Digital — 4-page Website (GitHub Ready)

## Pages
- `index.html` (Home)
- `services.html` (Services)
- `work.html` (Past Work)
- `consultation.html` (Request a Consultation)

## How to run locally
Just open `index.html` in your browser.

## Deploy on GitHub Pages (quick)
1. Create a new GitHub repo (example: `palmetto-digital-site`)
2. Upload all files from this folder to the repo root
3. In GitHub: Settings → Pages
4. Source: Deploy from a branch
5. Branch: `main` / folder: `/root`
6. Save → wait for GitHub to publish your site

## Enable the consultation form (optional)
This template uses Formspree:
- Create a Formspree form
- Replace the URL in `consultation.html`:
  `https://formspree.io/f/your-form-id`

## Swap the logo
Replace:
`assets/palmetto-digital-logo.png`
with your final transparent PNG when ready.
